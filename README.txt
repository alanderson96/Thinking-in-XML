How do you think XML relates to HTML? 
They're related in the fact that they both are mark up languages, and are similar in the fact that they both are simple. However, 
XML remains much more simple than HTML. 

What are some benefits of something like HTML using XML's syntax?  
HTML is a data processor and can take data from XML without compromising the data.

What are some disadvantages? 
Can be over-simplified. Can only be used for very specific functions.
